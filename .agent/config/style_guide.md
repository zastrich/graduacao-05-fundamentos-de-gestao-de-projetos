# Guia de Estilo

## Idioma
*   Português Brasileiro (pt-BR).
*   Linguagem técnica e objetiva.

## Markdown
*   Use headers hierarquicamente.
*   Destaque termos importantes em negrito.
*   Blocos de código sempre com linguagem especificada.

## Nomenclatura
*   Arquivos: kebab-case (ex: `introducao-ia.md`).
*   Diretórios: snake_case ou kebab-case consistente.
