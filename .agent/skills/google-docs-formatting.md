# Skill: Google Docs Advanced Formatting

This skill provides precise instructions for formatting academic documents in Google Docs using the Model Context Protocol (MCP) and browser interactions.

## Core Formatting Actions

### 1. Page Breaks
Instead of multiple empty lines, always use the explicit Page Break command.
- **Shortcut:** `Control+Enter`
- **Menu Path:** `Insert (Inserir)` -> `Break (Quebra)` -> `Page break (Quebra de página)`

### 2. Semantic Headings
To ensure an automatic Table of Contents works, you MUST apply semantic styles.
- **Heading 1 (Main Sections):** `Control+Alt+1`
- **Heading 2 (Subsections):** `Control+Alt+2`
- **Heading 3:** `Control+Alt+3`
- **Reset to Normal Text:** `Control+Alt+0` or `Control+\` (Clear formatting)

### 3. Automatic Table of Contents (TOC)
Never type a Table of Contents manually.
- **Action:** Use the search feature (`Alt+/` or clicking the search box) and type "Sumário".
- **Selection:** Choose the option with dots (e.g., "Inserir sumário com caracteres líderes de tabulação em pontos") for an academic look.
- **Update:** If sections change, click on the TOC in the document and click the "Atualizar sumário" (Refresh) icon.

### 4. Alignment & Indentation
- **Center Align:** `Control+Shift+E`
- **Left Align:** `Control+Shift+L`
- **Right Align:** `Control+Shift+R`
- **Justify:** `Control+Shift+J`

## Best Practices for Agents
1. **Search First:** If you can't find a menu item, use the `Alt+/` search box. It's the most reliable way to trigger specific features.
2. **Verify with Snapshots:** After applying a style, take a snapshot to ensure the "Styles" indicator in the toolbar reflects the change.
3. **Cursor Management:** Always `click` the editor area (aria-label="Conteúdo do documento") before typing or using shortcuts.
