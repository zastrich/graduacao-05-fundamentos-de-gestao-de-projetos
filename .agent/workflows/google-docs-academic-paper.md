# Workflow: Formatação de Trabalho no Google Docs

Este workflow descreve os passos para criar um documento formatado profissionalmente, incluindo capa, contra-capa, sumário automático e seções estruturadas.

## Passo 1: Preparação do Documento
1. Pressione `Control+A` e `Backspace` para limpar qualquer conteúdo anterior se necessário.
2. Defina a fonte padrão (ex: Arial ou Times New Roman, tamanho 12) via barra de ferramentas ou menu Formatar.

## Passo 2: Criação da Capa
1. Alinhe ao centro (`Control+Shift+E`).
2. Digite o cabeçalho (Instituição, Curso).
3. Pressione `Enter` várias vezes até o meio da página.
4. Digite o Nome do Autor (Negrito - `Control+B`).
5. Pressione `Enter` várias vezes.
6. Digite o Título do Trabalho (Negrito e tamanho maior, ex: 14 ou 16).
7. Pressione `Enter` várias vezes até o final da página.
8. Digite a Cidade e o Ano.
9. Insira uma quebra de página: `Control+Enter`.

## Passo 3: Criação da Contra-capa
1. Repita o Nome do Autor e o Título (Centralizado).
2. Pressione `Enter` algumas vezes.
3. Mude o alinhamento para a Direita (`Control+Shift+R`) ou use um recuo de parágrafo à direita para o texto descritivo do trabalho.
4. Digite a descrição do trabalho (ex: "Trabalho apresentado como requisito...").
5. Volte para o alinhamento Centralizado e digite Cidade e Ano no rodapé.
6. Insira uma quebra de página: `Control+Enter`.

## Passo 4: Sumário Automático
1. Digite "SUMÁRIO" (Heading 1 ou Normal Negrito Centralizado).
2. Pressione `Enter`.
3. Use a busca do Google Docs (`Alt+/`) para localizar "Inserir sumário com caracteres líderes de tabulação em pontos".
4. Selecione a opção. O sumário aparecerá vazio ou com as seções já formatadas.
5. Insira uma quebra de página: `Control+Enter`.

## Passo 5: Desenvolvimento do Conteúdo
1. Para cada seção principal, digite o título e aplique o Estilo **Título 1** (`Control+Alt+1`).
2. Para cada subseção, aplique o Estilo **Título 2** (`Control+Alt+2`).
3. O texto do corpo deve ser **Texto Normal** (`Control+Alt+0`) e preferencialmente Justificado (`Control+Shift+J`).

## Passo 6: Finalização
1. Volte à página do Sumário.
2. Clique no sumário e no botão de "Atualizar" para garantir que todas as seções e números de página estejam corretos.
