# Instruções dos Artefatos Práticos

Este diretório contém os artefatos visuais e documentais exigidos para a parte prática do trabalho.

## Conteúdo
1.  **Canvas de Visão [lean-vision-canvas.png](lean-vision-canvas.png):** Representação visual do modelo de negócio e estratégia do produto usando Miro.
2.  **Backlog [Backlog no Trello](https://trello.com/invite/b/6990eaa171fa0f1cf5a70e49/ATTIc97c3c88c31e016e3572fc185bf069b43F140572/ainsights) ou [cards-para-roadmap.md](cards-para-roadmap.md):** Lista detalhada e priorizada das funcionalidades para a primeira versão.
3.  **Matriz de Riscos [matriz-de-risco.png](matriz-de-risco.png) e [matriz-de-risco-detalhada.png](matriz-de-risco-detalhada.png):** Tabela de avaliação e plano de mitigação para os principais riscos identificados no produto, usando o Miro.
4.  **Roadmap [Board no Trello](https://trello.com/invite/b/6990eaa171fa0f1cf5a70e49/ATTIc97c3c88c31e016e3572fc185bf069b43F140572/ainsights) colocar visão Timeline ou [roadmap.png](roadmap.png):** Planejamento das 3 primeiras fases do produto.

## Como Interpretar
*   Comece pelo **Canvas** para entender a estratégia macro.
*   Analise o **Backlog** para ver o escopo tático do MVP ([definição do mvp](../teorico/2-mvp-definicao.md)) e próximos passos.
*   Verifique a **Matriz de Riscos** para entender o gerenciamento de incertezas.
*   Verifique o **Roadmap** para entender o planejamento das 3 primeiras fases do produto.
